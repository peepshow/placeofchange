    A GitHub theme for mdcss

    mdcss GitHub is a theme for mdcss based on the documentation styles seen across GitHub.

    Overview

    The template for this theme can be found in template.ejs, which contains the markup used to build out the logo, navigation, and main content. The styles you see around you can found inside assets/style.css. The primary scripting is handled in assets/script.js, but prism is used for syntax highlighting within code blocks, and assets/examples.color.js and assets/examples.html.js are used for special example code blocks.

    Logos

    The logo for this theme may be configured by setting the logo option in either in mdcss or this theme.

    require('mdcss')({
    theme: require('mdcss-theme-github')({
        logo: 'path/to/my/logo.svg'
    })
});

    Colors

    Color examples may be generated by using a color_example code block.

    @name: Firebrick @color: #ae1e23              @var: --red-firebrick
@name: Dark Fire @color: rgba(150,100,100,.8) @var: --red-dark-fire
@name: Platinum  @color: #e0e3e2              @var: --grey-platinum

    Typography

    This theme uses a common Helvetica font stack with a relative line height of 1.6 times the font size.

    Headings

    Headings match GitHub documentation conventions. In this theme, you can add _example to your code blocks to add a living example to your style guide.

    <h1>Heading Level 1</h1>
<h2>Heading Level 2</h2>
<h3>Heading Level 3</h3>
<h4>Heading Level 4</h4>
<h5>Heading Level 5</h5>
<h6>Heading Level 6</h6>

    Iconography

    This site has no icongraphy.

    Layout

    We use flex to horizontally stack the heading and main content areas. The navigation uses fixed position to stay on the screen as you scroll.
